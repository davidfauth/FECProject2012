Metadata Description

Data Dictionary - Contributions by Individuals

General Description

The individual contributions file contains each contribution from an individual to a federal committee if the contribution was at least $200.

Information contained in the list

The end-of-line (EOL) marker is line feed '\n' (LF, 0x0A, 10 in decimal). The Contributions by Individuals File contains the following information:
